# Simple HTML Introduction Page

This is a basic HTML webpage that introduces Hassan, a software engineering student at Mediacrest Training College.

## 📌 Project Description

The webpage displays:

- A short personal introduction
- Styled text using inline CSS
- External CSS file connection (`style.css`)

## 🛠 Technologies Used

- HTML5
- CSS3

## 📂 Project Structure


project-folder/
│
├── index.html
├── style.css
└── README.md


## 🚀 How to Run the Project

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html`.
4. The page will open in your web browser.

No installation or server required.

## 📖 Learning Purpose

This project demonstrates:

- Basic HTML structure
- Linking external CSS files
- Inline styling
- Paragraph formatting
